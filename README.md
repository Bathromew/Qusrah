# Qarah — Brand Experience Website

Qarah is a static, multi‑page marketing site for a creative/events studio. The site frames Qarah as “brand doctors,” showcasing services, featured projects, and team profiles with a strong visual system, custom typography, and motion-driven storytelling.

## Highlights
- 5-page layout: Home, Services, Portfolio, About, Contact
- Custom brand typography via local font files
- Animated hero typing effect, scroll reveals, and counters
- Interactive team cards with typed role/bio
- Fully responsive layout with mobile navigation
- Asset-rich design system (logos, patterns, icons)

## Pages
- `index.html`: Hero, brand scan CTA, featured brands, process
- `services.html`: Service/treatment cards and visual hero
- `portfolio.html`: Featured work grid
- `about.html`: Brand story and interactive team section
- `contact.html`: Contact form layout and contact details

## Tech Stack
- HTML5
- CSS3 (custom properties, responsive layouts, animations)
- Vanilla JavaScript (menu toggle, typing, scroll reveal, counters)
